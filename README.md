# CCE Skills Clinic - Login Page

## Overview

The CCE Skills Clinic is a support program that helps CCE students improve their academic performance through practical training and guidance. It provides enhanced and supportive services for learners of all academic levels.

This project is the **Login Page** for the CCE Skills Clinic system, created as a requirement for the 2nd Laboratory Exam.

---

## Features

- User login form with fields for username and password
- "Forgot Password?" link
- Footer with "Don’t have an account? Sign Up" and "Terms & Conditions"
- Responsive and clean design using HTML and CSS
- Session handling using PHP (demo with hardcoded credentials)

---

## Technologies Used

- HTML5
- CSS3
- PHP
